# Hackathon AMBEV – Análise de Dados

Este repositório contém a análise de dados desenvolvida para o case logístico da AMBEV.
O objetivo do projeto é analisar o impacto do aumento de demanda de produtos Long Neck, especialmente **Malzbier Brahma**, considerando limitações de produção e custos de transferências logísticas entre regiões.

A análise foi construída a partir dos dados disponibilizados em Excel e desenvolvida em **Python utilizando Jupyter Notebook**.

## Arquivo Principal

O notebook **`analise.ipynb`** contém a análise principal do projeto, incluindo tratamento dos dados, cálculos e visualizações utilizados para entender o cenário de demanda, produção e transferências.

Os arquivos dentro da pasta **`exploracao`** foram utilizados durante a etapa inicial para explorar e entender a estrutura das planilhas fornecidas no Excel.
